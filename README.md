# 💈 LaJack Booking System

Sistema de agendamento web desenvolvido sob medida para a **Barbearia LaJack**, visando substituir soluções genéricas por uma experiência personalizada e eficiente.

## 🚀 O Desafio
Otimizar o fluxo de agendamento, permitindo a inclusão de serviços adicionais (como sobrancelha) que sistemas padrão não gerenciam de forma fluida, garantindo a gestão correta do tempo do profissional.

## 🛠️ Tecnologias
- **Frontend:** Next.js, React, Tailwind CSS.
- **Backend:** Java + Spring Boot (ou Firebase).
- **Banco de Dados:** PostgreSQL / Firestore.

## 📈 Diferenciais do Projeto
- **UI/UX Customizada:** Identidade visual baseada na marca LaJack.
- **Add-on Logic:** Cálculo automático de tempo para serviços extras.
- **Mobile First:** Interface otimizada para agendamentos via smartphone.
